# 🔐 Ahfaz PDF Renamer  
### Offline Bulk PDF Renaming for Structured Document Workflows

Ahfaz PDF Renamer is an offline bulk PDF renaming tool that extracts text content from documents and automatically renames files based on detected information.

Designed for teams handling large volumes of text-based PDF documents who need structured, secure, and automated file renaming — without relying on cloud tools.

---

## 🚀 Key Features

- Extract text from text-based PDF files  
- Detect structured information inside documents  
- Apply customizable rename patterns  
- Bulk rename multiple PDFs instantly  
- 100% offline processing  
- No cloud dependency  
- Portable Windows application
